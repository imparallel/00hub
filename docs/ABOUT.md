# 00Hub — Personal Productivity Command Center

> *"작업실에 들어서면 가장 먼저 마주하는 단 하나의 화면."*

---

## 🧭 개요

**00Hub**는 ADHD 친화적 설계를 기반으로 한 **개인 생산성 대시보드**입니다.
복잡한 앱들 대신, 하루의 시작부터 끝까지 딱 하나의 창에서 **집중·행동·회고**를 동시에 관리할 수 있도록 설계되었습니다.

- 🏠 **운영 환경**: 로컬 실행
- 🎨 **디자인 철학**: CMYK 잉크 시스템을 재해석한 독자적 컬러 브랜딩
- 💾 **데이터**: 브라우저 LocalStorage
  → 모든 데이터는 브라우저 LocalStorage에 **초/개수 단위의 원시 데이터**로 정밀 저장
  → 데이터 유실 시 비율 기반의 **지능형 역산(Recovery)**을 통해 수치 복구
  → **v1.4.1+: ID 기반 실시간 중복 제거 필터 적용으로 데이터 무결성 보장**
- 🖥 **실행 방식**: `00Hub.exe` 더블 클릭 → 앱 모드로 즉시 실행
- ⌨️ **단축키**: `Alt + \`` (백틱) 키로 언제 어디서든 앱 표시/숨김 토글 (전역 단축키)
- 🔒 **고정**: 앱이 항상 최상단에 고정되어 Alt+Tab으로도 가려지지 않음

---

> **데이터 주의**: 브라우저 캐시를 완전히 삭제하면 데이터가 날아갈 수 있습니다. 가끔 하단 **Export** 버튼으로 백업을 권장합니다.

### 1. 📌 THE ONE THING — 오늘의 가장 중요한 것

화면 최상단을 가득 채우는 거대한 타이포그래피 입력창입니다.
오늘 **단 하나의 목표**를 적어두면, 스크롤 없이 항상 시야에 들어옵니다.
젠 모드에서도 동일한 텍스트가 그대로 표시되며, **자정이 지나면 자동으로 초기화**됩니다.

---

### 2. 🔥 Discipline Map — 30일 성취 히트맵

지난 30일간의 하루 성취도를 **CMYK 색상 레이어**로 시각화합니다.
단순한 비율 기록을 넘어, **초 단위 시간 및 항목 개수(Raw Data)**를 정밀하게 저장하여 목표치가 변해도 과거 데이터의 정확성을 유지합니다.

| 컬러 | 영역 | 측정 방식 (Precision Recording) |
| --- | --- | --- |
| 🟡 Yellow | Action Items | 완료 개수(Main/Sub) 정밀 기록 및 비율 환산 |
| 🔴 Magenta | Daily Quests | 일일 리셋 대비 완료/전체 개수 수학적 역산 복구 지원 |
| 🔵 Cyan | Focus Timer | 집중 시간(초 단위) 정밀 기록 (4시간 기준) |

셀에 마우스를 올리면 날짜와 각 영역별 **실제 수치(예: 1/3)**가 포함된 상세 툴팁이 표시됩니다.

---

### 3. ✅ Action Items — 행동 목록 & 슈퍼 보관함

**할 일을 관리하는 메인 컬럼**으로, 현재 집중할 일과 과거의 기록을 완벽히 분리해 관리합니다.

- **Active 탭 (집중)**:
  - 3단계 상태 순환: `TODO` → `DOING` → `DONE`
  - **인지 강화**: "접어두면 안 보게 된다"는 원칙에 따라 서브태스크가 상시 펼쳐져 있습니다.
- **Archive 탭 (슈퍼 보관함)**:
  - **날짜별 그룹화**: 완료된 날짜별로 항목들이 묶이며, 헤더를 클릭해 자유롭게 접고 펼 수 있습니다.
  - **기록 편집/삭제**: 이미 보관된 항목도 ✎ 버튼으로 수정하거나 ✕ 버튼으로 삭제할 수 있습니다.
  - **날짜 내 D&D**: 같은 날짜 안에서는 드래그 앤 드롭으로 순서를 변경할 수 있습니다.
  - **서브태스크 토글**: 보관함에서는 공간 효율을 위해 필요할 때만 서브태스크를 펼쳐봅니다.
- **공통**: 실수 방지를 위해 과거의 완료 항목은 🔒 아이콘과 함께 상태 변경이 잠깁니다.

---

### 4. ⭐ Daily Quests — 매일 반복 습관

**매일 리셋되는 반복 습관 체크리스트**입니다.

- 자정이 지나면 모든 퀘스트가 자동으로 미완료 상태로 초기화됩니다
- `✧` / `✦` 아이콘 클릭으로 완료/미완료 토글
- **드래그 앤 드롭**으로 우선순위 조정
- 새 퀘스트는 **맨 위에서 밀어내며** 추가됩니다
- 헤더에 `완료/전체` 진행률 표시 (완료 숫자는 **마젠타**, 나머지는 검은색으로 가독성 특화)
- 탭 크기와 정렬을 최적화하여 상단 제목 라인과 완벽한 시각적 균형 유지

---

### 5. ⏱ Focus Timer — 집중 시간 측정기

**오늘의 총 집중 시간**을 측정하는 타이머입니다.

- `Focus` 버튼으로 시작, `Pause`로 일시정지
- **작동 지연 표시**: 타이머 가동 시 제목 옆에 **스카이 블루(Cyan) 점**이 깜빡이며 작동 중임을 알립니다.
- `Reset`으로 오늘 집중 시간 초기화 (확인 필요)
- **자정이 지나면 자동으로 00:00:00으로 리셋**됩니다
- **목표 4시간 기준**으로 히트맵 Cyan 레이어에 반영
- **수직 중앙 정렬**: 모든 헤더 요소가 정중앙에 배치되어 시각적 안정감 제공

---

### 6. 🧠 Brain Dump — 머릿속 비우기

**아무 생각이나 흘려보내는 자유 메모장**입니다.

- 아이디어, 잡념, 나중에 할 것들을 적어 머리를 비웁니다
- 🗑 지우기 버튼으로 한 번에 초기화 (확인 필요)
- 내용은 자동 저장됩니다

---

### 7. 🧘 Zen Mode — 완전 집중 모드

시계 클릭 한 번으로 진입하는 **모든 것을 지우는 집중 화면**입니다.

- 배경, 카드, 모든 UI가 사라지고 **오직 목표 텍스트와 타이머만** 남습니다
- 진입 즉시 **백색소음**이 자동으로 재생됩니다 (별도 토글 버튼으로 조절 가능)
- **자연스러운 집중**: 젠 모드 진입 여부와 상관없이 앱은 항상 최상단에 고정되어 방해를 원천 차단합니다.
- Zen 전용 집중 타이머가 독립적으로 작동하며, 종료 시 메인 타이머에 합산됩니다
- 하단 중앙의 `✕` 버튼으로 메인 화면으로 복귀합니다

---

### 8. 💾 데이터 백업 / 복원

화면 하단 Footer에 위치한 두 버튼으로 관리합니다.

- **📤 Export Data**: 현재 모든 데이터를 날짜 이름의 JSON 파일로 다운로드
- **📥 Import Data**: 이전에 저장한 JSON 파일을 불러와 복원

> ⚠️ 데이터는 `00Hub.exe.WebView2` 폴더 내에 안전하게 저장되지만, 만약을 대비해 주기적으로 Export를 해두세요.

---

### 9. 🔔 프리미엄 알림 시스템

사용자 경험의 일관성을 위해 브라우저의 투박한 알림창을 전면 교체했습니다.

- **토스트 알림**: 성공, 경고, 잠금 안내 등 가벼운 피드백을 하단에 부드럽게 표시합니다.
- **확인 모달**: 데이터 삭제나 리셋 등 중요한 결정 시, 화면 중앙에 글래스모피즘 디자인의 전용 확인 창을 띄웁니다.
- 모든 알림은 `white-space: pre-line` 처리되어 긴 문장과 줄바꿈을 완벽하게 지원합니다.

## 🎨 디자인 시스템

00Hub의 모든 색상은 **인쇄 영역의 CMYK 4원색**에서 출발합니다.

```text
Yellow  #FEDA00  → 행동, 에너지, 진행 중인 것들
Magenta #FF0066  → 습관, 자기관리, 매일의 루틴
Cyan    #00B0FF  → 집중, 고요, 깊은 작업
Black   #121212  → 기반, 명확함, 타이포그래피
```

배경에는 세 가지 CMYK 색상이 희미한 **앰비언트 그라디언트**로 깔려 있어, 화면이 살아있는 느낌을 줍니다.
폰트는 `Outfit` (UI 본문)과 `JetBrains Mono` (타이머 숫자)를 사용합니다.

## ⚡ 실행 방법

```text
00Hub.exe 더블 클릭
  → npm run dev (Node.js 개발 서버) 백그라운드 실행
  → 2초 대기 (서버 안정화 대기)
  → WebView2 전용 창으로 앱 로드 (http://localhost:23500)
```

종료: `stop_hub.bat` 실행 → 서버 및 포트 완전 종료
